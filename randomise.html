About Clinical Trial Randomiser

Clinical Trial Randomiser is a tool to generate randomised list for conducting randomised control trials
Version 2, updated 25 June 2021
Copyright (C) 2020-2021 Goh Chan Sing

You can use this software for free for any legal purposes.
However, if you are using it in a research, it would be helpful if you cite this software in your paper!

  • Either in your methods (Example):
    The randomised list was generated using Clinical Trial Randomiser (Available from https://gohchansing.github.io/randomise.html), version 2
  • Or in your citations (Example):
    Goh, CS.(2020). Clinical Trial Randomiser (Version 2) [Software]. Available from https://gohchansing.github.io/randomise.html
